# Verkefni-6-Sk-rsla
Skýrsla

Höfundar: Jeremiah Páll Mascardo Patambag og Ólafur Ólafsson

Það eru tvö leikmenn með 1 takka sem virka eins og tening og báðir 4 reitir fyrir sig. Báðir ýta á takkana og leikmann sem er með hærra númer tekur yfir 1 reit. Báðir ýta á takkana þangað til hvor leikmenn vinna 8 reitirnar.
